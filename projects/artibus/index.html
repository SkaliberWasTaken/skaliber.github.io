ARTIBUS

Artibus is a brand new work in progress datapack that adds recipes, simple mechanics, mob customisation, additional dimensions, custom world generation, and additional structure spawning.

To follow the progress of the datapack as it's development progresses, you can follow me on My YouTube Channel.

Changelog

v0.7
  • Blocks of raw ore can be smelted into blocks of metal, saving on 1/9th fuel, but producing 1/9th less experience
  • Rooted dirt is now craftable
  • Added sawmill compatibility for all mangrove wood
  • Added special sawmill compatibility for mangrove roots
  • Re-added successful pack load announcement in chat
  • Re-added deaths scoreboard system
    •
  • Fixed vine loot table

v0.6
  • Removed all function based features
    •
  • Updated existed crafting tweaks to include 1.18 blocks and items

v0.5
  • Nether portals slowly infect the overworld, changing blocks to Netherrack, Soul Sand, and Blackstone
  • Zombies spawn twice as often on a full moon
  • Zombies drop Iron Nuggets at 10x the rate they previously dropped Iron Ingots
  • Smooth Stone is now crafted in the Stonecutter
  • Time will advance more quickly if every player is in the Nether (daytime extender feature)
  • Silk Touch can now be used to harvest Vines
  • Soul Sand can be smelted into Brown Glass
  • Soul Soil can be crafted into Coarse Dirt
  • Piglins are no longer aggravated by Players opening Ender Chests or Shulker Boxes
    •
  • Rotten Flesh now dries out to Rabbit Hide instead of Leather, but at 3x the speed
  • Zombies no longer drop Iron Ingots
  • Corrected the spelling on successful reload
  • Start up command to enable Player deaths
  • The Clock Advancement is now revoked from every Player on reload in case of stalling on installation

v0.4
  • Revamped phantom spawning mechanics
  • Pillagers hold a lantern in their off-hand during the night
  • Chains, end rods, and iron bars can be climbed
  • Enderman can spawn holding various blocks
  • Void world dimension
    •
  • Weather is now cleared when skipping the night
  • Changed the counter score on art_data to be hidden for visibility
  • Moved time store command to the tick
  • Endermen can pick up less blocks
  • Unique messages for initial startup, and reloads
  • Toolbox function to force first load
  • Load boolean which initiates startup functions
  • Rewritten the creative world startup function
  • Developer mode setting (useful for testing extended daytime related things)

v0.3

  • Pigs have a high chance to have a litter of 2-4 piglets when breeding
  • Rabbits have a high chance to have a litter of 2-6 kittens when breeding

  • Added recipe for green dye: craft blue and yellow dye together
    •
  • Adjustable multiplayer sleep system whilst daytime duration extender is disabled

  • Rabbits always drop 1 raw rabbit
  • Pigs drop slightly less porkchops than cows drop beef

v0.2
